# Portfolio CMS

Premium personal portfolio website with a private admin dashboard.

## Stack

- Frontend: React + Tailwind CSS + Vite
- Backend: Node.js + Express
- Database: MongoDB
- Media: Cloudinary
- Auth: JWT

## Apps

- `client`: public portfolio and admin CMS
- `server`: API, auth, database models, upload handling

## Run

```powershell
Set-Location "C:\portfolio-cms"
npm.cmd install
npm.cmd run install:all
npm.cmd run dev
```

## URLs

- Client: `http://localhost:5173`
- Server: `http://localhost:5000`
"# portfolio"
